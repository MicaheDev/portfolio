Use of JavaScript:
I used JavaScript to add three interactive features: a dark/light theme toggle that persists the user's preference in localStorage, a function to show/hide the navigation menu on mobile devices, and simulated alert functions for the policy and help links.

Use of Bootstrap:
I integrated Bootstrap primarily for the grid system using utility classes like 'row' and 'col-lg-6', and for design utilities like 'd-flex', 'text-center', and 'px-4' to manage layout and spacing. I also used Bootstrap Icons for visual elements.
